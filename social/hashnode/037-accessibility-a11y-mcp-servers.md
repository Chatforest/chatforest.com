**At a glance:** 20+ servers across 5 subcategories — full-page auditing, color contrast, WCAG remediation, enterprise integration, and color blindness simulation.

Accessibility MCP servers bring automated WCAG compliance testing into AI coding workflows. AI agents scan pages, check contrast, validate ARIA, and fix issues through MCP.

## Full-Page Auditing

Three competing axe-core scanners dominate:

- **[a11ymcp](https://github.com/ronantakizawa/a11ymcp)** (78 stars, 6 tools) — 5,000+ downloads, WCAG 2.0/2.1/2.2, HTML snippet testing
- **[mcp-accessibility-scanner](https://github.com/JustasMonkev/mcp-accessibility-scanner)** (43 stars, 15+ tools) — Playwright multi-page crawling, keyboard nav, matrix scanning
- **[a11y-mcp](https://github.com/priyankark/a11y-mcp)** (40 stars, 2 tools) — agentic loop: audit, fix, re-audit

## Color Contrast

Three servers solve LLM contrast ratio hallucination: ryelle's a11y-color-contrast-mcp, bryanberger's batch analysis tool, and AccessLint's server with color suggestions.

## What's Good

Strong web page auditing. BrowserStack official (130 stars, Spectra™ engine). WCAG remediation with 10-tool server (HTML/React/Vue/Angular). 57+ accessibility agents from Community-Access project.

## What's Not

No mobile testing (VoiceOver/TalkBack). No PDF remediation. No screen reader emulation. 10+ servers doing variations of the same scan.

## Rating: 3.5/5

Excellent tooling for web auditing, but narrow focus. Three competing scanners plus BrowserStack enterprise coverage make the audit-fix-reaudit loop well-supported.
