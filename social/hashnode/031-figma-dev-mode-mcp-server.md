**At a glance:** 443 stars, 13 tools, OAuth, remote at mcp.figma.com/mcp, code-to-canvas in 7+ IDEs, Framelink competitor at 13.8K stars

Figma's official MCP server bridges design and code. Paste a Figma URL, ask your AI to build it. The standout: **code-to-canvas capture** — build in code, push back to Figma as an editable design. No community server can do this.

## What It Does

8 read tools (design context, variables, Code Connect, screenshots, metadata) and 5 write tools (code-to-canvas, Code Connect mapping, FigJam diagrams, design system rules). Write operations bypass rate limits.

## Official vs Framelink

Framelink (community, 13.8K stars) produces better output for pure design-to-code: descriptive JSON, preserved nesting, ~25% smaller payloads, no seat-type rate limits. But only the official server has code-to-canvas, Code Connect, and design system rules.

Free tier: **6 calls/month** (official) vs Figma API limits only (Framelink).

## Rating: 3.5/5

Unique write capabilities no community server can match. Loses points for the crippling free tier, prescriptive React/Tailwind defaults, and flattened component hierarchies. Paid Figma plan effectively required.
