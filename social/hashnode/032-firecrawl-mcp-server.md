**At a glance:** 95,700+ parent stars, 5,800+ MCP server stars, 12+ tools, FIRE-1 agent (beta), Browser Sandbox, ~50.6K weekly PulseMCP visitors

Firecrawl is a web data platform with an MCP server. Scrape pages, crawl sites, search the web, extract structured data with LLM, run autonomous research, control cloud browsers — all through MCP tools.

## Key Tools

- **Scraping** — single page, batch, async crawling, URL mapping
- **Search & extraction** — web search with geo targeting, LLM-powered JSON extraction
- **Research** — autonomous agent browsing and multi-source deep research
- **Browser** — persistent cloud sessions with CDP access

## What's Good

Autonomous deep research and LLM extraction are genuinely unique. Production-grade scraping with JS rendering and anti-bot. Self-hostable. Massive 95.7K-star ecosystem.

## What's Not

500 one-time free credits. Credit stacking (1 credit becomes 9 with extraction). MCP server hasn't been updated since September 2025 while the platform shipped major features. Overkill for simple web reading — free Fetch MCP handles 80% of use cases.

## Rating: 4/5

Most comprehensive web scraping MCP server available. Unmatched tool breadth. But the MCP server lags the platform, pricing is complex, and free alternatives cover most basic needs.
