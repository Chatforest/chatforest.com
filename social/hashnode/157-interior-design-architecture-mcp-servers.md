**At a glance:** Every major CAD platform now has MCP support. Blender MCP leads with 17.6k stars — one of the most popular MCP servers in existence. Rhino/Grasshopper has 5+ implementations. FreeCAD-MCP offers 57 tools with Docker headless execution. The glaring gap: actual interior design tools don't exist yet. **Rating: 4/5.**

## Blender

### ahujasid/blender-mcp (17,600+ stars)

[blender-mcp](https://github.com/ahujasid/blender-mcp) (Python, 18+ tools). Two-way AI-Blender communication — object manipulation, material control, geometry nodes, asset libraries (Polyhaven/Sketchfab), Hyper3D model generation, viewport screenshots, and arbitrary Python execution. At 17.6k stars, this is one of the most popular MCP servers across all categories.

## AutoCAD

- **[autocad-mcp](https://github.com/puran-water/autocad-mcp)** (159 stars, Python, 8 tools) — freehand AutoLISP execution, 600+ ISA 5.1-2009 P&ID symbols, ezdxf headless backend (no AutoCAD needed)
- **[multiCAD-mcp](https://github.com/AnCode666/multiCAD-mcp)** (46 tools) — supports AutoCAD, ZWCAD, GstarCAD, BricsCAD with batch operations
- **CAD-MCP**, **mcp-autocad-server** — additional natural language CAD control options

## FreeCAD

### proximile/FreeCAD-MCP (57 tools)

[FreeCAD-MCP](https://github.com/proximile/FreeCAD-MCP) — the highest tool count of any CAD MCP server. Docker headless execution, multi-AI support (Claude/GPT-4o/Gemini), TRELLIS.2 image-to-3D generation, Vision AI analysis, Gradio web interface, and Cloudflare tunnels.

Also: bonninr/freecad_mcp (direct Claude interaction), lucygoodchild/freecad-mcp-server (geometry/booleans), spkane/freecad-addon-robust-mcp-server (MCP Bridge Workbench), contextform/freecad-mcp.

## SketchUp

**[sketchup-mcp](https://github.com/mhyrr/sketchup-mcp)** (182 stars, Ruby/Python) — component manipulation, materials, scene inspection, Ruby code evaluation, and unique **woodworking joinery tools** (mortise/tenon, dovetail, finger joints).

## Rhino & Grasshopper (5+ servers)

- **[rhinomcp](https://github.com/jingcheng-chen/rhinomcp)** (274 stars) — object manipulation, layer management, script execution, JSON TCP protocol
- **GH_mcp_server** — direct Rhino+Grasshopper interaction, GHPython generation
- **rhino-grasshopper-mcp** — ML-based automatic layout optimization
- **grasshopper-mcp** — component creation with high-level intent recognition
- **rhino_mcp** — prompt-assisted 3D modeling

## Revit / BIM

**[revit-mcp](https://github.com/revit-mcp/revit-mcp)** (18 stars, C#/Python) — the only major BIM platform with MCP support. CRUD operations on Revit elements, plugin architecture, extensible command sets.

## Fusion 360 & OpenSCAD

- **Fusion-360-MCP-Server** — thread-safe API, works with Claude/ChatGPT
- **fusion360-mcp-server** — script generation and execution
- **[openscad-mcp](https://github.com/quellant/openscad-mcp)** — multi-perspective rendering, animation, 300 tests, 80%+ coverage

## What's Missing

No dedicated interior design servers (room layout, furniture placement, color palettes), no floor plan generators, no rendering engines (V-Ray, Lumion, Enscape), no landscape architecture, no building code compliance, no material catalogs (IKEA, Wayfair), no SolidWorks or Onshape.

## Bottom Line

Outstanding CAD/3D tool coverage with strong community adoption — every major platform has MCP support and Blender alone has 17.6k stars. But the interior design workflow side (room layout, furniture, colors) is entirely missing.

**Rating: 4/5**
