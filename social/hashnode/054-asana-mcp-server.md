**At a glance:** Hosted at `mcp.asana.com/v2/mcp` — OAuth 2.0, Streamable HTTP, ~44 tools, 20 featured client integrations. V2 launched Feb 2026, V1 shutdown May 11. The most tool-rich productivity MCP server we've reviewed. **Rating: 4/5.**

## What It Does

**Task management (8 tools)** — Create, update, delete, search tasks. Advanced search across text, dates, status, assignee, projects, portfolios, sections, tags, custom fields.

**Projects and portfolios (12 tools)** — Create projects, list sections, track progress, manage portfolios, color-coded health updates.

**Goals and time management (7 tools)** — OKRs, goal hierarchies, progress metrics. No other productivity MCP server offers goal tracking.

**Dependencies (4 tools)** — Prerequisites, blocking relationships, follower management.

**Team management (5+ tools)** — Workspaces, teams, users, resource allocations.

**Collaboration (5 tools)** — Comments, activity history, attachments.

## What Sets It Apart

- **Most complete PM MCP server** — 44 tools vs Todoist (37+), Linear (23+)
- **Hosted zero-install** with OAuth 2.0
- **20 client integrations** — Claude, ChatGPT, Cursor, VS Code, Figma, and more
- **Permission inheritance** — actions use authenticated user's permissions

## What's New (March 2026)

Asana AI Teammates launched March 17 — 21 pre-built AI agents. Claude-to-Asana integration ships. $6M ARR from AI products.

## What's Missing

V2 still lacks comments/stories (V1 shuts down May 11). Single-workspace sessions (regression from V1). No self-hosted option.

## Community Alternative

**roychri/mcp-server-asana** (131 stars, MIT) — 40+ tools, PAT auth, has comments that official V2 still lacks.

**Rating: 4/5** — Most tool-rich productivity MCP server with proper hosted architecture. Incomplete V2 transition and single-workspace limitation keep it from 4.5.
