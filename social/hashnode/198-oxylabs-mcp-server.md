Part of our **[Web Search & Data Extraction MCP category](https://chatforest.com/categories/web-search-scraping/)**.

**At a glance:** 86 stars, MIT license, v0.7.5 (December 2025), Python, 8 tools across 2 engines, two separate free trials available.

Oxylabs takes a different approach: instead of one unified tool set, it exposes **two distinct scraping engines** through a single MCP interface. The company has $43.7M in revenue (2025), 428 employees, and a proxy network covering 195+ countries.

## What It Does

### Web Scraper API Tools (4 tools)

| Tool | Purpose |
|------|---------|
| **universal_scraper** | Scrape any URL with anti-bot bypass |
| **google_search_scraper** | Structured Google Search results |
| **amazon_search_scraper** | Amazon search pages |
| **amazon_product_scraper** | Individual Amazon product data |

Routes through Oxylabs' proxy network — IP rotation, CAPTCHA handling, JavaScript rendering, HTML-to-Markdown conversion.

### AI Studio Tools (4 tools)

| Tool | Purpose |
|------|---------|
| **ai_scraper** | AI-powered content extraction (JSON/Markdown) |

| **ai_browser_agent** | Remote browser automation |
| **ai_search** | Web search with AI-powered extraction |

## Benchmark Performance

In [AIMultiple's independent benchmarks](https://aimultiple.com/browser-mcp):

| Server | Accuracy | Avg Speed | Scalability (250 agents) |
|--------|----------|-----------|--------------------------|
| Bright Data | 100% | 30s | 76.8% |
| Nimble | 93% | 16s | 51.2% |
| Firecrawl | 83% | 7s | 64.8% |
| **Oxylabs** | **75%** | **14s** | **54.4%** |

Fastest stress test completion: **31.7 seconds average** (vs. Bright Data 48.7s, Nimble 182.3s). Speed is the story, not raw accuracy.

## The Bottom Line

**Rating: 3/5** — The dual-engine approach is genuinely interesting — traditional proxy scraping + AI-powered extraction in one package. Fastest stress test times (31.7s) show solid infrastructure. Low entry point ($12/mo for AI Studio).

But 75% accuracy puts Oxylabs in the bottom half. Just 86 stars and a four-month-old release. The 8-tool surface is thin vs. Bright Data (60+) or Firecrawl (19). Good for teams already using Oxylabs' proxy infrastructure. For everyone else, Bright Data (reliability) or Firecrawl (speed + open source) are stronger picks.
