Publishing and typesetting MCP servers give AI agents the ability to compile LaTeX documents, convert between document formats, manage eBook libraries, create print-on-demand products, and automate desktop publishing workflows.

This review covers 55+ servers across LaTeX/Overleaf, document conversion, eBooks, book discovery, print-on-demand, desktop publishing, and PDF tools. ## Document Format Conversion — The Strongest Subcategory

| Server | Stars | Tools | Key Feature |
|--------|-------|-------|-------------|
| [zcaceres/markdownify-mcp](https://github.com/zcaceres/markdownify-mcp) | 2,400 | 10 | Convert PDF/DOCX/XLSX/PPTX/images/audio/YouTube to Markdown |
| [vivekVells/mcp-pandoc](https://github.com/vivekVells/mcp-pandoc) | 512 | 1 | Pandoc-based conversion: Markdown/HTML/PDF/DOCX/LaTeX/EPUB/RST/ODT |
| [2b3pro/markdown2pdf-mcp](https://github.com/2b3pro/markdown2pdf-mcp) | 29 | 1 | Markdown to PDF with Mermaid diagrams and watermarks |

**markdownify-mcp** (2,400 stars) is the most popular server in the entire category — the universal ingest pipeline for getting documents into AI workflows. **mcp-pandoc** (512 stars) is the only server offering true multi-format output including EPUB generation.

## LaTeX & Overleaf

| Server | Stars | Tools | Key Feature |
|--------|-------|-------|-------------|
| [mjyoo2/OverleafMCP](https://github.com/mjyoo2/OverleafMCP) | 73 | 6 | Read-only Overleaf Git integration |

| [takashiishida/arxiv-latex-mcp](https://github.com/takashiishida/arxiv-latex-mcp) | 111 | 4 | Fetch arXiv paper LaTeX source directly |

The LaTeX space is fragmented — 5+ competing Overleaf implementations. **arxiv-latex-mcp** (111 stars) solves a real pain point for academic researchers by fetching LaTeX source instead of PDF. **TeXFlow** takes an elegant approach where AI works with semantic structure while the system handles all LaTeX mechanics.

**ebook-mcp** (351 stars) is surprisingly comprehensive — smart library management, content discovery, interactive AI-powered reading with quizzes, and natural language navigation.

## Print-on-Demand & Desktop Publishing

**Printify MCP** (24 stars) enables end-to-end AI product creation with integrated image generation via Replicate's Flux model. **Lulu Print MCP** provides 20+ tools for the Lulu Print API. InDesign servers (135+ tools) are technically impressive but require macOS + Adobe InDesign.

## What's Missing

- No EPUB authoring tools (only reading)
- No Amazon KDP integration
- No Typst MCP server (the modern LaTeX alternative)
- No book cover design tools
- No Affinity Publisher integration

## Rating: 3.5/5

Document conversion is genuinely excellent — markdownify-mcp and mcp-pandoc are best-in-class infrastructure. eBook management is unexpectedly mature. LaTeX workflows are well-served despite fragmentation. But desktop publishing is platform-limited, the self-publishing pipeline has too many gaps, and the absence of Amazon KDP integration is the most glaring omission.
