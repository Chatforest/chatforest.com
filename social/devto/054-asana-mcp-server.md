**At a glance:** Hosted at `mcp.asana.com/v2/mcp` — OAuth 2.0, Streamable HTTP, ~44 tools, 20 featured client integrations. V2 launched Feb 2026, V1 shutdown May 11. The most tool-rich productivity MCP server we've reviewed. **Rating: 4/5.**

## What It Does

The 44 tools cover the full Asana hierarchy:

**Task management (8 tools)** — Create, update, delete, search tasks. Advanced search across text, dates, status, assignee, projects, portfolios, sections, tags, and custom fields.

**Projects and portfolios (12 tools)** — Create projects, list sections, track progress statistics, manage portfolios, document project health with color-coded updates.

**Goals and time management (7 tools)** — Create and track OKRs, navigate goal hierarchies, update progress metrics. No other productivity MCP server offers goal tracking through MCP.

**Dependencies and relationships (4 tools)** — Prerequisite and blocking relationships, follower management.

**Team and user management (5+ tools)** — Workspace discovery, team structure, user lookup, resource allocations.

**Collaboration (5 tools)** — Task comments, activity history, attachment access.

## What Sets It Apart

- **Most complete PM MCP server** — 44 tools vs Todoist (37+), Linear (23+), Atlassian (undocumented)
- **Hosted zero-install** — No npm, no Docker, just connect and authorize
- **Permission inheritance** — All actions execute with authenticated user's permissions
- **20 client integrations** — Claude, Claude Code, ChatGPT, Codex, Cursor, VS Code, Windsurf, Figma, Zoom, Perplexity, and more

## What's New (March 2026)

- **Asana AI Teammates** launched March 17 — 21 pre-built AI agents (Campaign Brief Writer, Bug Investigator, etc.)
- **Claude-to-Asana integration** — Turn Claude conversations into Asana projects/tasks
- **$6M ARR from AI products** — Targeting 15% of FY27 new ARR from AI

## What's Missing

- **V2 tool gaps** — Comments/stories still missing as of early March. V1 shuts down May 11.
- **Single-workspace sessions** — V2 restricts to one workspace at a time (regression from V1)
- **No self-hosted option** — Hosted-only at mcp.asana.com
- **Agent parameter issues** — Some reports of null parameters with smaller models

## Community Alternative

**roychri/mcp-server-asana** (131 stars, MIT) — 40+ tools via npm, Personal Access Tokens, local stdio. Has comments/stories that official V2 still lacks. Read-only mode available.

## How It Compares

- **vs. Todoist** — Todoist is better for personal task management; Asana wins on organizational depth (goals, portfolios, workload)
- **vs. Linear** — Linear is engineering-focused; Asana handles cross-functional enterprise work
- **vs. Atlassian** — Atlassian covers Jira + Confluence; Asana has more tools and broader client compatibility

**Rating: 4/5** — The most tool-rich productivity MCP server with proper hosted architecture. The incomplete V2 transition, single-workspace limitation, and Asana's pricing ceiling keep it from 4.5.
